# Bicycle Sales Analysis

An international bicycle company has compiled datasets of customers' details, product description, and sales. We are tasked to create a database and perform analysis. Datasets include birthdata of customers, categories and cost of products, and total sales for each purchases of a product.

A SQL database management system called duckdb is being used with Jupyter notebook as interface. Six csv files are uploaded in a dataset folder and are merged into three files, of which a database are created.
