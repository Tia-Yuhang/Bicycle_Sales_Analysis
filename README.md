# Bicycle Sales Analysis

An international bicycle company has compiled datasets of customers' details, product descriptions, and sales. We are tasked to create a database and perform analysis. Datasets include customer birthdate, product cost and category, and sales for each customer order.

A SQL database management system called duckdb is being used with Jupyter notebook as interface. Six csv files are uploaded in a dataset folder and are merged into three files, of which a database are created.
